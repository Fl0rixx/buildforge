# BuildForge

Сайт для подбора и сравнения комплектующих ПК.

## Возможности

- Конфигуратор ПК
- Автоматический расчёт стоимости
- Проверка совместимости комплектующих
- Сравнение процессоров и видеокарт
- Готовые сборки для разных задач
- FAQ с популярными вопросами
- Адаптация под мобильные устройства

## Сайт

https://fl0rixx.github.io/buildforge/

## О проекте

BuildForge помогает подобрать комплектующие для игрового или рабочего компьютера, сравнить характеристики и оценить итоговую стоимость сборки.
